A modern web application for EEG signal analysis built with Vite, React, and Plotly.js.

**Location:** `eeg-insights-react/`

- Mock LLM-generated insights
- Markdown and JSON report export

**Quick Start:**
```bash
cd eeg-insights-react
npm install
npm run dev
```
